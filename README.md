##  Технічна складова та архітектура

При розробці проєкту було враховано ключові вимоги до сучасної фронтенд-розробки:

* **HTML5 (Семантична структура):** Чиста та валідна розмітка сторінок із використанням правильних тегів семантики (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`).
* **CSS3 (Адаптивність та Анімація):** Дизайн сайту повністю адаптований під мобільні пристрої, планшети та десктопи завдяки технологіям **Flexbox** (для навігації та хедера) та **CSS Grid** (для карток персонажів). Додано плавні CSS-анімації (`@keyframes`) для ефектної появи контенту.
* **JavaScript & jQuery:** Логіка інтерфейсу поєднує чистий JavaScript (робота з DOM, валідація) та бібліотеку jQuery для створення динамічних елементів без перезавантаження сторінок.

##  Реалізований інтерактивний функціонал

1.  **SPA-навігація (Система Табів):** На головній сторінці реалізовано плавне перемикання між фракціями гри (*CyberLife*, *Єрихон*, *Поліція DPD*) за допомогою jQuery-методів `.fadeIn()` та `.hide()`, що дозволяє вивчати лор без перезавантаження браузера.
2.  **Динамічний конструктор модулів (Робота з DOM):** На сторінці персонажів інтегровано кастомний трекер, який дозволяє користувачу в реальному часі створювати комплектацію андроїда — динамічно додавати нові програмні пакети та видаляти їх зі списку за допомогою чистого JS.
3.  **Імітація AJAX-запиту (Форма зворотного зв'язку):**
    * Форма "Стати частиною повстання" містить клієнтську валідацію (перевірка коректності Email за символом `@` та мінімальної довжини імені).
    * При успішному заповненні полів імітується асинхронна затримка відповіді сервера (`setTimeout`), після чого виводиться модальне вікно зі статусом фіксації девіантності на 100%.
